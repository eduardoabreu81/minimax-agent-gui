# 設定

設定パネル（サイドバー下部）は、アカウント、モデル、外観を構成する場所です。

## アカウントと API

MiniMax アカウントを接続し、アプリがプラットフォームにアクセスするために使う API 構成を管理します。プラン（Plus / Max / Ultra）はここで検出され、利用できる機能を制御します。

## モデル

利用可能なモデルを確認します。新しいメッセージの既定モデルはステータスバーで選びますが、モデル関連の設定はここにあります。

## 外観

ライトテーマとダークテーマを切り替えます。一部のテーマには、オン・オフできる追加の視覚効果が含まれます。

## 言語

インターフェースは複数の言語で利用できます。言語を変更すると、翻訳がある場合はこのヘルプの内容も対応する言語に切り替わり、ない場合は英語に戻ります。

## プロジェクトコンテキスト

エージェントコンテキストの仕組みを使うと、プロジェクトを一度説明するだけで、エージェントが必要な背景を持てます。セットアップウィザードまたはコンテキストバナーから開けます。
