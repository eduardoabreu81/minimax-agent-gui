# キーボードショートカット

| ショートカット | 操作 |
| --- | --- |
| `Ctrl / Cmd + K` | コマンドパレットを開く |
| `F1` または `?` | ヘルプを開く |
| `Enter` | 現在のメッセージを送信 |
| `Shift + Enter` | コンポーザーで改行 |
| `/` | スラッシュコマンド（コンポーザー先頭） |
| `@` | コンテキスト参照を添付 |

## コマンドパレット

コマンドパレット（`Ctrl/Cmd + K`）は、移動と操作を最速で行う方法です。キーボードから手を離さずに、任意のパネルへ移動したり、新しいチャットを始めたり、設定を開いたり、テーマを切り替えたりできます。
