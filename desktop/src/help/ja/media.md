# メディアスタジオ

メディアパネルは、MiniMax の生成モデルを使ってプロンプトを画像・動画・音楽・音声に変換します。それぞれサイドバーに独自の項目があります。

## 画像

欲しい画像を説明して生成します。文字数カウンターでプロンプトを上限内に保ち、結果をダウンロードまたは再利用します。

## 動画

プロンプトから短い動画クリップを生成します。動画生成は最も計算負荷の高い機能で、上位プランに制限される場合があります。

## 音楽

スタイル、雰囲気、楽器構成のテキスト説明から音楽トラックを生成します。

## 音声（TTS）

テキストを音声に変換します。ボイスを選んでナレーションや返答を生成します。

## クレジット

メディア生成はクレジットを消費します。サイドバーのクレジットウィジェットに現在の残高が表示され、各生成の完了後に更新されます。
