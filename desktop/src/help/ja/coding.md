# コードエージェント

コードエージェント（**コード**パネル）は自律型のコーディング作業環境です。チャットとは異なり、実際のワークスペースでファイルの読み書きやコマンドの実行ができ、複数ステップのエンジニアリング作業を遂行します。

## 仕組み

やりたいこと——機能、修正、リファクタリング——を説明すると、エージェントが計画し、ファイルを編集し、コマンドを実行して実現します。進捗はライブでストリーミングされ、取り組んでいるステップを示す ToDo リストが表示されます。

## ターミナル

統合ターミナルには、エージェントが実行するコマンドとその出力が表示されます。実際のバックエンドに対して動作するため、シミュレーションではなく実際の実行を反映します。

## モデルと思考

チャットと同様に、コードエージェントはステータスバーのモデルと拡張思考の設定を使います。思考を有効にした `MiniMax-M3` は複雑な作業に最も強力な選択肢です。

## ヒント

- 望む成果と制約を具体的に伝えます。
- ストリーミングされるエージェントの計画と差分を確認します。
- 作業は焦点を絞って——小さく適切に範囲を定めた依頼ほど確実に動作します。
