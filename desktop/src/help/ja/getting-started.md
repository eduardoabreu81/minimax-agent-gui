# はじめに

MiniMax Studio は MiniMax プラットフォーム向けのデスクトップ作業環境です。エキスパートモデルとチャットし、自律型コーディングエージェントを実行し、メディアを生成し、作業を管理する——これらすべてを 1 つのウィンドウで行えます。

## 初回起動

アプリを起動すると、ローカルバックエンドが利用可能になるのを待ち（短い読み込み画面）、**チャット**パネルが開きます。セットアップウィザードが表示された場合は、MiniMax アカウントの接続とプロジェクトコンテキストの入力を案内します。

## 作業環境の概要

- **サイドバー**（左）— チャット、コード、画像、動画、音楽、音声、タスクを切り替えます。シェブロンで折りたたんでスペースを確保できます。
- **タイトルバー**（上）— アプリの識別表示とコマンドパレットのボタン。
- **ステータスバー**（下）— アクティブなモデルと拡張思考の切り替え。ここで選んだものが次のメッセージで使われます。
- **設定とヘルプ**（サイドバー下部）— アカウント、モデル、テーマ、そしてこのヘルプ。

## モデルの選択

ステータスバーのモデルセレクターで、次のメッセージを処理するモデルを選びます。拡張思考は `MiniMax-M3` で利用でき、対応していないモデルでは切り替えが無効になります。

## 次のステップ

- **チャット**パネルを開いて最初のメッセージを送ります。
- 実践的なコーディング作業には**コードエージェント**を試します。
- どこからでも `Ctrl/Cmd + K` でコマンドパレットを開けます。
