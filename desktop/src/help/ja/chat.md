# チャット

チャットパネルは MiniMax のエキスパートモデルとの会話です。質問、下書き、分析など、ファイルシステムに触れる必要のない作業に使います。

## メッセージの送信

下部のコンポーザーに入力し、`Enter` で送信します（`Shift + Enter` で改行）。モデルは応答をトークンごとにストリーミングします。

## コンポーザーの機能

- **スラッシュコマンド** — コンポーザーの先頭で `/` を入力するとコマンドメニューが開きます。
- **@ 参照** — `@` を入力して、コンテキスト（ファイルやその他の参照）をチップとしてメッセージに添付します。
- **添付** — プロンプトと一緒にモデルに読ませるファイルを追加します。

## モデルと思考

アクティブなモデルと拡張思考の切り替えはウィンドウ下部のステータスバーにあり、コードエージェントと共有されます。ここで切り替えると、どちらのパネルでも次のメッセージに影響します。

## やり直す

コマンドパレット（`Ctrl/Cmd + K`）で**新しいチャット**を選ぶと、会話をクリアして最初からやり直せます。
