# 聊天

聊天面板是与 MiniMax 专家模型的对话。用它来提问、起草、分析，以及任何无需触及文件系统的工作。

## 发送消息

在底部的编辑器中输入并按 `Enter` 发送（`Shift + Enter` 换行）。模型会逐个 token 流式返回回复。

## 编辑器功能

- **斜杠命令**——在编辑器开头输入 `/` 打开命令菜单。
- **@ 引用**——输入 `@` 将上下文（文件或其他引用）以标签形式附加到消息。
- **附件**——添加文件，让模型与你的提示一起读取。

## 模型与思考

当前模型和扩展思考开关位于窗口底部的状态栏，并与代码智能体共享——在此切换会影响任一面板的下一条消息。

## 重新开始

使用命令面板（`Ctrl/Cmd + K`）并选择**新建聊天**，即可清空对话并重新开始。
