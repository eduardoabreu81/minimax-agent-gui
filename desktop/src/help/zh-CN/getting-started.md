# 快速开始

MiniMax Agent 是面向 MiniMax 平台的桌面工作区——与专家模型聊天、运行自主编码智能体、生成媒体并跟踪工作，全部在一个窗口中完成。

## 首次启动

应用启动时会等待本地后端就绪（短暂的加载界面），然后打开**聊天**面板。如果出现设置向导，它会引导你连接 MiniMax 账户并填写项目上下文。

## 工作区一览

- **侧边栏**（左侧）——在聊天、代码、图像、视频、音乐、语音和任务之间切换。用箭头折叠以腾出空间。
- **标题栏**（顶部）——应用标识和命令面板按钮。
- **状态栏**（底部）——当前模型和扩展思考开关。你在此处的选择会用于下一条消息。
- **设置与帮助**（侧边栏底部）——账户、模型、主题以及本帮助。

## 选择模型

使用状态栏中的模型选择器来选择处理下一条消息的模型。扩展思考在 `MiniMax-M3` 上可用；不支持的模型会禁用该开关。

## 后续步骤

- 打开**聊天**面板并发送第一条消息。
- 试用**代码智能体**来完成实际的编码任务。
- 在任何位置按 `Ctrl/Cmd + K` 打开命令面板。
