# 代码智能体

代码智能体（**代码**面板）是一个自主编码工作区。与聊天不同，它可以在真实的工作区中读写文件并运行命令，以完成多步骤的工程任务。

## 工作原理

描述你想要的——一个功能、一处修复、一次重构——智能体会规划、编辑文件并运行命令来实现它。其进度会实时流式显示，并用待办列表展示它正在处理的步骤。

## 终端

集成终端会显示智能体运行的命令及其输出。它针对真实后端运行，因此反映的是实际执行而非模拟。

## 模型与思考

与聊天一样，代码智能体使用状态栏中的模型和扩展思考设置。启用思考的 `MiniMax-M3` 是处理复杂任务的最强选择。

## 提示

- 明确说明你想要的结果以及任何约束。
- 在智能体的计划和差异（diff）流式出现时进行查看。
- 让任务保持聚焦——更小、范围更明确的请求运行得更可靠。
