# 设置

设置面板（侧边栏底部）是你配置账户、模型和外观的地方。

## 账户与 API

连接你的 MiniMax 账户，并管理应用访问平台所用的 API 配置。你的订阅层级（Plus / Max / Ultra）在此处检测，并决定哪些功能可用。

## 模型

查看可用模型。新消息的默认模型在状态栏中选择，但与模型相关的偏好设置在此处。

## 外观

在浅色和深色主题之间切换。某些主题包含可开关的额外视觉效果。

## 语言

界面提供多种语言。更改语言时，若存在翻译，本帮助内容也会切换为相应语言，否则回退到英文。

## 项目上下文

智能体上下文系统让你只需描述一次项目，智能体便拥有所需的背景信息。你可以从设置向导或上下文横幅中打开它。
