# Chat

O painel Chat é uma conversa com os modelos especialistas da MiniMax. Use-o
para perguntas, rascunhos, análises e tudo que não precise tocar no seu
sistema de arquivos.

## Enviando uma mensagem

Digite no compositor na parte de baixo e pressione `Enter` para enviar
(`Shift + Enter` insere uma nova linha). O modelo transmite a resposta token
a token.

## Recursos do compositor

- **Comandos de barra** — digite `/` no início do compositor para abrir o
  menu de comandos.
- **Referências com @** — digite `@` para anexar contexto (arquivos ou outras
  referências) à sua mensagem como chips.
- **Anexos** — adicione arquivos para o modelo ler junto com o seu prompt.

## Modelo e raciocínio

O modelo ativo e o botão de raciocínio estendido ficam na barra de status, no
rodapé da janela, e são compartilhados com o Agente de Código — trocar aqui
afeta a próxima mensagem em qualquer um dos painéis.

## Começando do zero

Use a paleta de comandos (`Ctrl/Cmd + K`) e escolha **Novo chat** para limpar
a conversa e recomeçar.
