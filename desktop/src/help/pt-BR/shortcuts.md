# Atalhos de Teclado

| Atalho | Ação |
| --- | --- |
| `Ctrl / Cmd + K` | Abrir a paleta de comandos |
| `F1` ou `?` | Abrir a Ajuda |
| `Enter` | Enviar a mensagem atual |
| `Shift + Enter` | Nova linha no compositor |
| `/` | Comandos de barra (início do compositor) |
| `@` | Anexar uma referência de contexto |

## Paleta de comandos

A paleta de comandos (`Ctrl/Cmd + K`) é a forma mais rápida de navegar e
executar ações — pule para qualquer painel, inicie um novo chat, abra as
configurações ou troque o tema sem tirar as mãos do teclado.
