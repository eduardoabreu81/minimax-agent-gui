# Configurações

O painel de Configurações (rodapé da barra lateral) é onde você configura sua
conta, modelos e aparência.

## Conta e API

Conecte sua conta MiniMax e gerencie a configuração de API que o app usa para
acessar a plataforma. Seu plano (Plus / Max / Ultra) é detectado aqui e
controla quais recursos ficam disponíveis.

## Modelos

Revise os modelos disponíveis. O modelo padrão para novas mensagens é
escolhido na barra de status, mas as preferências relacionadas a modelos
ficam aqui.

## Aparência

Alterne entre os temas claro e escuro. Alguns temas incluem efeitos visuais
extras que você pode ativar ou desativar.

## Idioma

A interface está disponível em vários idiomas. Mudar o idioma também troca o
conteúdo desta ajuda para o idioma correspondente, quando há tradução,
voltando para o inglês caso contrário.

## Contexto do projeto

O sistema de Contexto do Agente permite descrever seu projeto uma vez para
que o agente tenha o histórico de que precisa. Você pode abri-lo pelo
assistente de configuração ou pelo banner de contexto.
