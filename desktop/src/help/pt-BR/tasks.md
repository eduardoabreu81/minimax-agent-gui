# Quadro de Tarefas

O Quadro de Tarefas (painel **Tarefas**) acompanha itens de trabalho em um
quadro simples para você ver o que está planejado, em andamento e concluído.

## Trabalhando com tarefas

Crie tarefas, mova-as entre as colunas conforme o trabalho avança e use a
barra de estatísticas para ver as contagens de relance.

## Por que ele existe

Trabalhos longos do agente e projetos de várias etapas ficam mais fáceis de
acompanhar quando as etapas estão visíveis. O quadro dá um lugar persistente
para registrar isso, separado de qualquer conversa específica.
