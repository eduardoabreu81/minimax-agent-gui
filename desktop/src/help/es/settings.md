# Configuración

El panel de Configuración (pie de la barra lateral) es donde configuras tu
cuenta, modelos y apariencia.

## Cuenta y API

Conecta tu cuenta de MiniMax y gestiona la configuración de API que la app usa
para acceder a la plataforma. Tu plan (Plus / Max / Ultra) se detecta aquí y
controla qué funciones están disponibles.

## Modelos

Revisa los modelos disponibles. El modelo predeterminado para los mensajes
nuevos se elige en la barra de estado, pero las preferencias relacionadas con
los modelos están aquí.

## Apariencia

Alterna entre los temas claro y oscuro. Algunos temas incluyen efectos
visuales extra que puedes activar o desactivar.

## Idioma

La interfaz está disponible en varios idiomas. Cambiar el idioma también
cambia el contenido de esta ayuda al idioma correspondiente, cuando hay
traducción, y vuelve al inglés en caso contrario.

## Contexto del proyecto

El sistema de Contexto del Agente te permite describir tu proyecto una vez
para que el agente tenga el contexto que necesita. Puedes abrirlo desde el
asistente de configuración o el banner de contexto.
