# Primeros Pasos

MiniMax Studio es un espacio de trabajo de escritorio para la plataforma
MiniMax: chatea con modelos expertos, ejecuta un agente de programación
autónomo, genera contenido multimedia y haz seguimiento de tu trabajo, todo
en una sola ventana.

## Primer inicio

Al arrancar, la app espera a que el backend local esté disponible (una breve
pantalla de carga) y abre en el panel **Chat**. Si aparece un asistente de
configuración, te guía para conectar tu cuenta de MiniMax y completar el
contexto de tu proyecto.

## El espacio de un vistazo

- **Barra lateral** (izquierda): alterna entre Chat, Código, Imagen, Vídeo,
  Música, Voz y Tareas. Contráela con la flecha para ganar espacio.
- **Barra de título** (arriba): identidad de la app y el botón de la paleta de
  comandos.
- **Barra de estado** (abajo): el modelo activo y el interruptor de
  razonamiento extendido. Lo que elijas aquí es lo que usa el siguiente
  mensaje.
- **Configuración y Ayuda** (pie de la barra lateral): cuenta, modelos, tema y
  esta ayuda.

## Elegir un modelo

Usa el selector de modelo en la barra de estado para elegir qué modelo atiende
tu siguiente mensaje. El razonamiento extendido está disponible en
`MiniMax-M3`; el interruptor se desactiva para modelos que no lo admiten.

## Próximos pasos

- Abre el panel **Chat** y envía tu primer mensaje.
- Prueba el **Agente de Código** para tareas prácticas de programación.
- Pulsa `Ctrl/Cmd + K` para abrir la paleta de comandos desde cualquier lugar.
