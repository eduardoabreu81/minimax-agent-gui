# Atajos de Teclado

| Atajo | Acción |
| --- | --- |
| `Ctrl / Cmd + K` | Abrir la paleta de comandos |
| `F1` o `?` | Abrir la Ayuda |
| `Enter` | Enviar el mensaje actual |
| `Shift + Enter` | Nueva línea en el compositor |
| `/` | Comandos de barra (inicio del compositor) |
| `@` | Adjuntar una referencia de contexto |

## Paleta de comandos

La paleta de comandos (`Ctrl/Cmd + K`) es la forma más rápida de navegar y
ejecutar acciones: salta a cualquier panel, inicia un nuevo chat, abre la
configuración o cambia el tema sin soltar el teclado.
