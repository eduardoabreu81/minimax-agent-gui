# 채팅

채팅 패널은 MiniMax 전문가 모델과의 대화입니다. 질문, 초안 작성, 분석 등 파일 시스템을 건드릴 필요가 없는 작업에 사용합니다.

## 메시지 보내기

아래 작성기에 입력하고 `Enter`로 보냅니다(`Shift + Enter`는 줄바꿈). 모델은 응답을 토큰 단위로 스트리밍합니다.

## 작성기 기능

- **슬래시 명령** — 작성기 맨 앞에서 `/`를 입력하면 명령 메뉴가 열립니다.
- **@ 참조** — `@`를 입력해 컨텍스트(파일이나 기타 참조)를 칩 형태로 메시지에 첨부합니다.
- **첨부** — 프롬프트와 함께 모델이 읽을 파일을 추가합니다.

## 모델과 사고

활성 모델과 확장 사고 토글은 창 하단 상태 표시줄에 있으며 코드 에이전트와 공유됩니다. 여기서 전환하면 두 패널 중 어느 쪽이든 다음 메시지에 영향을 줍니다.

## 새로 시작

명령 팔레트(`Ctrl/Cmd + K`)에서 **새 채팅**을 선택하면 대화를 지우고 처음부터 다시 시작할 수 있습니다.
