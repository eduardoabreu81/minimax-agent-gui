# 시작하기

MiniMax Studio는 MiniMax 플랫폼을 위한 데스크톱 작업 공간입니다. 전문가 모델과 채팅하고, 자율 코딩 에이전트를 실행하고, 미디어를 생성하고, 작업을 추적하는 일을 모두 하나의 창에서 할 수 있습니다.

## 첫 실행

앱을 시작하면 로컬 백엔드가 준비될 때까지 기다린 뒤(짧은 로딩 화면) **채팅** 패널이 열립니다. 설정 마법사가 나타나면 MiniMax 계정 연결과 프로젝트 컨텍스트 입력을 안내합니다.

## 작업 공간 한눈에 보기

- **사이드바**(왼쪽) — 채팅, 코드, 이미지, 비디오, 음악, 음성, 작업 사이를 전환합니다. 화살표로 접어 공간을 확보할 수 있습니다.
- **제목 표시줄**(위) — 앱 식별 표시와 명령 팔레트 버튼.
- **상태 표시줄**(아래) — 활성 모델과 확장 사고 토글. 여기서 선택한 것이 다음 메시지에 사용됩니다.
- **설정 및 도움말**(사이드바 하단) — 계정, 모델, 테마, 그리고 이 도움말.

## 모델 선택

상태 표시줄의 모델 선택기로 다음 메시지를 처리할 모델을 선택합니다. 확장 사고는 `MiniMax-M3`에서 사용할 수 있으며, 지원하지 않는 모델에서는 토글이 비활성화됩니다.

## 다음 단계

- **채팅** 패널을 열고 첫 메시지를 보냅니다.
- 실습형 코딩 작업에는 **코드 에이전트**를 사용해 보세요.
- 어디서나 `Ctrl/Cmd + K`로 명령 팔레트를 열 수 있습니다.
