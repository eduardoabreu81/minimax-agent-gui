# 키보드 단축키

| 단축키 | 동작 |
| --- | --- |
| `Ctrl / Cmd + K` | 명령 팔레트 열기 |
| `F1` 또는 `?` | 도움말 열기 |
| `Enter` | 현재 메시지 보내기 |
| `Shift + Enter` | 작성기에서 줄바꿈 |
| `/` | 슬래시 명령(작성기 맨 앞) |
| `@` | 컨텍스트 참조 첨부 |

## 명령 팔레트

명령 팔레트(`Ctrl/Cmd + K`)는 이동하고 동작을 실행하는 가장 빠른 방법입니다 — 키보드에서 손을 떼지 않고 어떤 패널로든 이동하고, 새 채팅을 시작하고, 설정을 열고, 테마를 전환할 수 있습니다.
