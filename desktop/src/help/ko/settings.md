# 설정

설정 패널(사이드바 하단)은 계정, 모델, 외관을 구성하는 곳입니다.

## 계정 및 API

MiniMax 계정을 연결하고 앱이 플랫폼에 접속하는 데 사용하는 API 구성을 관리합니다. 요금제(Plus / Max / Ultra)가 여기서 감지되어 사용 가능한 기능을 제어합니다.

## 모델

사용 가능한 모델을 확인합니다. 새 메시지의 기본 모델은 상태 표시줄에서 선택하지만, 모델 관련 환경설정은 여기에 있습니다.

## 외관

밝은 테마와 어두운 테마를 전환합니다. 일부 테마에는 켜고 끌 수 있는 추가 시각 효과가 포함됩니다.

## 언어

인터페이스는 여러 언어로 제공됩니다. 언어를 변경하면 번역이 있는 경우 이 도움말 내용도 해당 언어로 전환되며, 없으면 영어로 돌아갑니다.

## 프로젝트 컨텍스트

에이전트 컨텍스트 시스템을 사용하면 프로젝트를 한 번 설명해 두어 에이전트가 필요한 배경을 갖추게 할 수 있습니다. 설정 마법사나 컨텍스트 배너에서 열 수 있습니다.
