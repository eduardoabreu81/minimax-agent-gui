# 미디어 스튜디오

미디어 패널은 MiniMax 생성 모델을 사용해 프롬프트를 이미지, 비디오, 음악, 음성으로 변환합니다. 각 항목은 사이드바에 별도로 있습니다.

## 이미지

원하는 이미지를 설명하고 생성합니다. 글자 수 카운터로 프롬프트를 제한 내로 유지하고, 결과를 다운로드하거나 재사용합니다.

## 비디오

프롬프트로 짧은 비디오 클립을 생성합니다. 비디오 생성은 가장 연산 부담이 큰 기능으로, 상위 요금제로 제한될 수 있습니다.

## 음악

스타일, 분위기, 악기 구성에 대한 텍스트 설명으로 음악 트랙을 생성합니다.

## 음성(TTS)

텍스트를 음성 오디오로 변환합니다. 보이스를 선택해 내레이션이나 응답을 생성합니다.

## 크레딧

미디어 생성은 크레딧을 소비합니다. 사이드바의 크레딧 위젯에 현재 잔액이 표시되며 각 생성이 완료될 때마다 갱신됩니다.
