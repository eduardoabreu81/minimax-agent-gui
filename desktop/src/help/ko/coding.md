# 코드 에이전트

코드 에이전트(**코드** 패널)는 자율 코딩 작업 공간입니다. 채팅과 달리 실제 작업 공간에서 파일을 읽고 쓰며 명령을 실행해 여러 단계의 엔지니어링 작업을 수행할 수 있습니다.

## 작동 방식

원하는 것——기능, 수정, 리팩터링——을 설명하면 에이전트가 계획을 세우고 파일을 편집하고 명령을 실행해 이를 달성합니다. 진행 상황이 실시간으로 스트리밍되며, 진행 중인 단계를 보여주는 할 일 목록이 표시됩니다.

## 터미널

통합 터미널에는 에이전트가 실행하는 명령과 그 출력이 표시됩니다. 실제 백엔드를 대상으로 동작하므로 시뮬레이션이 아니라 실제 실행을 반영합니다.

## 모델과 사고

채팅과 마찬가지로 코드 에이전트는 상태 표시줄의 모델과 확장 사고 설정을 사용합니다. 사고를 켠 `MiniMax-M3`는 복잡한 작업에 가장 강력한 선택입니다.

## 팁

- 원하는 결과와 제약 조건을 구체적으로 알려주세요.
- 스트리밍되는 에이전트의 계획과 차이(diff)를 검토하세요.
- 작업의 초점을 좁게 유지하세요 — 더 작고 범위가 명확한 요청일수록 더 안정적으로 실행됩니다.
